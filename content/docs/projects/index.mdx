## 在建项目

文件结构

```
content/docs/projects/
├── index.mdx
├── anatomy-tw.mdx
├── mri-pm-zen.mdx
├── todolist.mdx
└── meta.json
```

meta.json

```json
{
  "title": "在建项目",
  "pages": ["anatomy-tw", "mri-pm-zen", "todolist"]
}
```

<Callout>不要在meta.json中添加`index.mdx`，这样folder就可以正确指向`index.mdx`。</Callout>
